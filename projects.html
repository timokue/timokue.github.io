Robotics & Microcontroller

The Development of a Self-driving, object-avoiding robot with Nvidia Jetson and ultrasonic sensors. This project was about developing a self-driving robot that can explore its environment and avoid obstacles using ultrasonic sensors and a camera controlled by an Nvidia Jetson board. To achieve this goal, we used ultrasonic sensors and multiple Attiny microcontrollers to detect obstacles and communicate with the Jetson board via the I2C bus. The Jetson board processed the sensor data and used it to control the robot's movement and steering.

AI

Handwritten Character Recognition with AI. The goal of this project, was to develop an AI system that uses JavaScript and tensors to recognize handwritten numbers and characters and accurately classify them. The AI model was trained with the MNIST dataset, a large dataset of handwritten digits. The AI model learns to recognize and classify various handwritten digits and characters by analyzing the patterns and features of each image. The AI model was integrated into a website where users can draw their own handwritten characters and numbers using a mouse or touchpad. The AI model then analyzes the image and classifies the handwritten character or number. If the AI model misclassifies the user's input, the user can select the correct character or number, and the AI model learns from this correction to improve its accuracy.

Generative Art

Creating generative artworks with P5.js. The goal of this project was to create generative artworks using code and P5.js, a JavaScript library for creative coding. The project is an exploration of the possibilities of combining art and code to create unique, visually stunning artworks. Using P5.js, a series of generative artworks were created, experimenting with different techniques and algorithms. In particular, the influence of randomly generated parameters, provided impressive results.
